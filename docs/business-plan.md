## 第一章 · 执行摘要

### 1.1 一句话定位

> **为赴日动漫朝圣游客打造的「一站式御宅族旅行地图」**——覆盖 **176家** 动漫店铺、**7大连锁** 品牌、内置 AI 旅行助手，用 AutoSec 零信任安全架构守护用户数据。

> **长期愿景**：从「二次元店铺地图」进化为「中国人去日本旅游的第一张地图」——六阶段扩展至购物、景点、美食、住宿四大品类 **1,500+** 地点，2028年H1目标MAU **30万**。

### 1.2 核心数据摘要

| 指标 | 数值 |
|------|------|
| 已收录店铺 | **176家**（东京 89 / 大阪 42 / 名古屋 21 / 其他 24） |
| 覆盖连锁品牌 | **7个**（Animate / Mandarake / Toranoana / K-Books / Lashinbang / Surugaya / Melonbooks） |
| 目标市场 TAM | **¥2,160亿日元**（动漫相关旅游消费） |
| 首年目标用户 | **50,000** MAU |
| 首年营收目标 | **¥180万 RMB** |
| 盈亏平衡点 | **第18个月** |
| 融资需求 | **¥50万 RMB**（种子轮） |

### 1.3 产品核心优势

1. **垂直深度**：不是通用地图加几个标记，而是为御宅族游客量身定做——包含店铺楼层信息、营业时间、特色商品分类、限定品信息
2. **AI 导游**：DeepSeek 驱动的智能助手，理解动漫语境，能推荐"离你最近的高达模型店"或"秋叶原同人志路线"
3. **安全零信任**：AutoSec 138条规则引擎，从前端到 API 全链路防护，保护用户位置和搜索隐私
4. **完全开源底座**：React 19 + MapLibre GL JS 5 + OpenFreeMap，无 Google Maps 高额 API 费用

### 3.3 目标用户画像

| 画像 | 特征 | 需求 | 痛点 |

### 3.4 市场机会窗口

1. **签证放宽**：2024年中国赴日签证简化，游客数量V型反弹
2. **日元贬值**：日元兑人民币处于历史低位（2026年约¥5.2 = ¥100日元），刺激消费型旅游
3. **IP联动热**：《咒术回战》《间谍过家家》《葬送的芙莉莲》等IP催生圣地巡礼热潮
4. **垂直化需求**：通用地图无法满足"找特定类型店铺"的深度需求
5. **竞品空白**：目前无直接竞品占据"动漫购物地图"这一赛道

## 第十二章 · 风险分析

### 12.1 风险矩阵

#### 风险总览

| 风险类型 | 概率 | 影响 | 等级 |
|------|------|------|------|
| 市场风险 | 中 | 高 | 高 |
| 竞争风险 | 中 | 中 | 中 |
| 技术风险 | 低 | 高 | 中 |
| 数据风险 | 高 | 中 | 中 |
| 资金风险 | 中 | 高 | 高 |
| 合规风险 | 低 | 高 | 中 |
| 团队风险 | 低 | 高 | 中 |

#### 风险描述与缓解措施

**市场风险**（等级：高）
- 描述：赴日旅游政策突变（疫情/签证/地缘政治）
- 缓解：多国扩展（韩/泰）；开发"云逛店"线上功能

**竞争风险**（等级：中）
- 描述：Google Maps/anitabi增加类似功能
- 缓解：持续深耕数据深度；社区网络效应护城河

**技术风险**（等级：中）
- 描述：DeepSeek API不稳定或大幅涨价
- 缓解：多模型冗余（Qwen/GLM备选）；热门回答本地缓存

**数据风险**（等级：中）
- 描述：店铺信息过时/闭店未更新
- 缓解：用户纠错机制；月度巡检；店铺官方数据通道

**资金风险**（等级：高）
- 描述：融资失败/资金链断裂
- 缓解：极低burn rate；提前启动变现；Bootstrap备案

**合规风险**（等级：中）
- 描述：日本APPI/中国个保法合规问题
- 缓解：AutoSec合规模块；日本法务顾问；最小数据收集

**团队风险**（等级：中）
- 描述：核心成员离开/精力分散
- 缓解：4年Vesting绑定；代码文档化；尽早补充人才
